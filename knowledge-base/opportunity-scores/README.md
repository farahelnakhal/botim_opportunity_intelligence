# knowledge-base/opportunity-scores/

Owned by Workstream B (Product & Opportunity Intelligence).

Holds completed scorecards, one per proposition, from `opportunity-intelligence/frameworks/opportunity-scoring.md`.

## Contents

- `<idea-slug>.md` — all 17 individual 1–5 scores with basis (evidence ID or `(A)` assumption), evidence confidence, main assumptions, main invalidation risk, cross-module dependency, recommended next action, and the indicative composite (shown last, one decimal).

## Rules

- All 17 scores are always shown; a composite alone is never a valid scorecard.
- Re-score when new evidence lands; keep prior scores in a dated history section so score drift is visible.
- If more than 6 scores are `(A)`, the linked proposition cannot be classified above "Promising but unvalidated".
