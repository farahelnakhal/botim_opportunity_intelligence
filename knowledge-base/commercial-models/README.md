# knowledge-base/commercial-models/

Owned by Workstream B (Product & Opportunity Intelligence).

Holds completed commercial models and subsidy models, one set per proposition.

## Contents

- `<idea-slug>.md` — full commercial model (from `opportunity-intelligence/templates/commercial-model.md`): volume drivers, revenue/cost lines, downside/base/upside, break-even, assumption register.
- `<idea-slug>-subsidy.md` — MDR/interchange subsidy model (from `templates/mdr-interchange-subsidy-model.md`): net payment margin, max affordable free-credit days / cashback / fee subsidy.
- `ASSUMPTIONS.md` (optional, once models accumulate) — shared assumption registry (interchange rates, funding rate, ECL) so models stay consistent; update in one place.

## Rules

- Every figure is a range with F/E/A labelling. Never model BOTIM as receiving full MDR.
- When an assumption is validated or refuted by an experiment, update the model and note the date.
