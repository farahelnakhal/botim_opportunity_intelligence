# Opportunity Profile Template

The consolidated evaluation form: every field of the opportunity framework in one fillable file, so profiles are complete and comparable rather than free-form. One file per proposition: `knowledge-base/product-ideas/<opp-slug>.md`. Mark every unevidenced value `(A)`; cite evidence as `EV-…`.

---

## OPP-nnn — <name>

**Classification:** Strong opportunity / Promising but unvalidated / Weak / Reject (from stress test; engine enums: strong/promising/weak/reject)
**Evidence confidence:** High / Medium / Low · **Linked artefacts:** scorecard JSON · model inputs JSON · computed model · VE specs/results · recommendation
