### Field guide

- **Type:** review-store / forum / subreddit / social / vendor-changelog / help-centre / feature-board / podcast / case-study / search-demand / community-group / complaint-site / aggregator / other
- **Access:** direct / api / search-snippet / rss / archived / aggregator / public-index / licensed / manual-collection-needed / blocked-lawfully-inaccessible
- **Yield:** high / medium / low / dead-end — how much usable evidence it has produced
- **Quality notes:** independence, authenticity concerns (e.g. review bursts), survivorship caveats

### Rules

- Append-only: never delete a row; mark dead sources `dead-end` and move on.
- `manual-collection-needed` rows must include exact instructions for a human (where to look, what to capture).
- `blocked-lawfully-inaccessible` means paywall/auth/robots.txt prevented access — record the lawful alternative attempted.
- Re-check high-yield sources on every weekly update; note the date in **Last checked**.
